# 🏥 Government Healthcare Resource Intelligence Platform

## AI-Powered Decision Support System for Public Healthcare Planning

An end-to-end **Data Analytics & Machine Learning** platform that helps government authorities identify healthcare infrastructure gaps, prioritize districts, and support data-driven healthcare resource allocation across India.

The platform combines **interactive analytics dashboards**, **AI-powered healthcare priority prediction**, and **executive decision support** into a single Streamlit application.

## 📸 Project Overview

The platform transforms healthcare datasets into actionable insights using advanced analytics and machine learning.

### Key Objectives

* Analyze healthcare infrastructure across India
* Identify high-priority healthcare districts
* Monitor maternal and child health indicators
* Evaluate vaccination coverage
* Detect healthcare accessibility gaps
* Predict Healthcare Priority Score using AI
* Support government decision-making with interactive dashboards

### 🤖 AI Healthcare Prediction

Machine Learning powered prediction using **XGBoost Regression**

Predicts Healthcare Priority Score using:

* Population
* Health Centres
* Population per Health Centre
* Literacy
* Female Literacy
* Vaccination
* Anaemia
* Stunted Children
* Underweight Children
* Institutional Birth
* Antenatal Care

Outputs include:

* Healthcare Priority Score
* Priority Classification
* Interactive Gauge Chart
* AI Recommendations
* Feature Importance
* Downloadable Prediction Report
